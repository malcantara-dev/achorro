## Achorro - Encontre seu pet!

Projeto criado para a disciplina de `Desenvolvimento Web` durante o curso de `Engenharia de Computação` pela `FACENS - Faculdade de Engenharia de Sorocaba`.

Este sistema tem como objetivo conectar pessoas que procuram por animais de estimação perdidos ou fugidos com aquelas que avistaram pets nas ruas, oferecendo uma plataforma para cadastrar, buscar e visualizar informações sobre animais perdidos e achados.

## Tecnologias Utilizadas

- **Next.js**: Framework React para construção da interface e API
- **PostgreSQL**: Banco de dados relacional utilizado para armazenar as informações de usuários e pets
- **Prisma**: ORM para manipulação do banco de dados
- **bcrypt**: Para criptografar senhas dos usuários
- **NextAuth.js**: Biblioteca para autenticação de usuários

## Funcionalidades

- Cadastro de usuários e pets
- Login e logout de usuários
- Busca e filtro de pets disponíveis para adoção
- Visualização de informações detalhadas de cada pet
- Sistema de autenticação utilizando NextAuth.js
